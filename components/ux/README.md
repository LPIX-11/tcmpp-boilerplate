# UX Components

This directory is for UX-focused components (onboarding flows, empty states, error boundaries, loading skeletons).

## Conventions

- Each component follows the 4-file pattern: `index.js`, `index.json`, `index.wxml`, `index.wxss`
- Prefix: `app-ux-` (e.g., `<app-ux-empty-state>`)
